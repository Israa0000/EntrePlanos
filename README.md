## Como hacer un merge
1. UNITY TIENE QUE ESTAR CERRADO 
2. Cada uno en su rama pa hacer y comitesr cambios.
3. Unity cerrao
4. cambiar a rama activa dev
5. merge de la rama tuya a dev
6. abrir unity pa ver que to funcione como debería
7. unity cerrao
8. commit
9. cambiar a rama activa la de ca uno
10. una vez haya cambios significativos en dev, merge de dev a main
